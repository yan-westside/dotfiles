Always ask clarifying questions BEFORE saving memory — do not save until answers are confirmed.

**Why:** Saves wrong/incomplete information otherwise, which then pollutes future sessions.

**How to apply:**
1. When user provides new information or classification, first ask any clarifying questions needed to fully understand it
2. Only after user confirms answers → save to memory
3. Only after memory is saved → proceed with the actual work
4. This applies to all new data, classifications, business rules, or process changes
