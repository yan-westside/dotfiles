Richard Lam (Finance) prefers the following format for AOP vs current HC comparisons:

**Time horizon**: Weekly for current quarter, **monthly averages for H2** (Jul–Dec 2026).

**Delta thresholds for flagging material changes:**
- **In-house**: ±5 HC per Site-LOB (e.g., PHX-Dx Global Advocates)
- **BPO**: |diff / base| ≥ 5% per Px-Queue (percentage, not flat count)

**Why**: Richard uses these comparisons to track staffing ramps vs AOP for Finance planning. He wants H2 monthly averages to see the full ramp trajectory, not just a point-in-time snapshot.

**How to apply**: When sending Richard HC comparison data, always include:
1. Current quarter weekly view (already in weekly Finance update)
2. H2 monthly averages table (Jul/Aug/Sep/Oct/Nov/Dec) for both in-house and BPO
3. Flag any LOB/queue where delta exceeds ±5 vs AOP
